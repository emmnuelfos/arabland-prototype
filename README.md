## Try the interactive features

From the **Homepage**, scroll down and try:

- **Heart icon** on any listing — adds it to the shortlist drawer (top-right counter)
- **Compare checkbox** on listing cards — up to 4 properties pin to a bottom tray
- **Click a property card** — opens the full property detail modal with photo gallery, mortgage calculator, schedule-a-viewing
- **"Open the map"** strip — opens the map preview with property pins
- **Off-Plan section "View payment plan"** — shows the installment timeline visualizer
- **Top-right toggles** — currency (AED ⇄ USD ⇄ EUR), area unit (sqft ⇄ sqm), language (EN / AR)
- **"Tweaks" panel** (bottom-right corner of the homepage) — switch the hero variant, listing card style, display font, and toggle the valuation strip live

## Brand language

- **Warm Ochre** `#AC7B43` — primary accent (used sparingly)
- **Graphite** `#4D4B4A` — dominant dark
- **Stone Grey** `#B2B4B2` — hairlines & secondary text
- **Porcelain** `#EAE8E3` — page background

Type stack: **Cormorant Garamond** (display serif) · **Macan Pan** (body) · **JetBrains Mono** (eyebrows).

## Status

This is a **design prototype** — clickable but not connected to a real CMS, CRM, or live data. Every listing, agent, and project shown is sample data. The production build wires this to:

- **Goyzer CRM** — real-time property/agent sync, ContactInsert lead capture
- **GenieMap** — interactive plot map for off-plan project pages
- **Dubai Land Department** — DLD-verified badges (subject to API permissions)
- **Mortgage gateway** — pre-approval submissions
- **GA4 + GTM + Meta Pixel** — full conversion tracking
